# 🜁 Doable

**Doable** — веб-приложение для задач, целей, привычек и дневника.

![[./assets/images/screenshot.png]]

## 🚀 Основной функционал

### ⚰️ Ежедневные задачи (День)

- Добавление, удаление, выполнение задач
- Переключение между днями (стрелки + клик по заголовку)
- Автоопределение «сегодня» с учётом правила 2 часов ночи
- Фильтр выполненных задач
- Добавление задачи на следующий день: `Shift + Enter` / `Shift + клик по +`
- Редактирование задачи: двойной клик по тексту
- Выделение важной задачи: навести курсор + `Alt + Q`
- Сохранение в `localStorage`

### 🜃 Ежемесячные цели (Месяц)

- Приоритеты: `1_` (высокий), `2_` (средний), `3_` (низкий)
- Сортировка: сначала невыполненные по приоритету
- Переключение месяцев (стрелки + клик по заголовку)
- Фильтр выполненных целей
- Добавление цели в текущий день: клик по `+`
- Добавление цели в следующий день: `Shift + клик по +`
- Редактирование цели: двойной клик (приоритет сохраняется)
- Выделение важной цели: навести курсор + `Alt + Q`

### 🜄 Календарь

- Сетка текущего месяца
- Индикатор задач (точка) и записей в дневнике
- Подсветка текущего дня `—today`
- Подсветка выбранной даты из дневника `—active`
- Клик по дню → переход в дневник

### 🜅 Дневник

- Поля: эмоции 😶, события 🕯️, цели 🜏
- Автосохранение при потере фокуса
- Календарь для навигации
- Копирование заметки в буфер
- Очистка заметки
- Переход на сегодня: `Alt + T` (работает и внутри)

### ⚰️ Трекер привычек

- Добавление/удаление привычек
- Отметка выполнения по дням
- Адаптивное количество дней: 5 на мобильных устройства (<700px), 7 на ПК
- Режим редактирования: карандашик → крестики
- Редактирование названия: двойной клик в режиме редактирования

### 🎨 Темы оформления

- Тёмная (по умолчанию) и светлая
- Переключение: кнопка в навигации / `Alt + E`
- Сохранение темы в `localStorage`
- Плавные переходы цветов

### ⌨️ Горячие клавиши

| Шорткат                     | Действие                             |
| --------------------------- | ------------------------------------ |
| `Alt + E`                   | Сменить тему                         |
| `Alt + T`                   | Открыть дневник / перейти на сегодня |
| `Alt + Q`                   | Выделить задачу/цель под курсором    |
| `Alt + D`                   | Фокус на поле ввода задач дня        |
| `Alt + M`                   | Фокус на поле ввода целей месяца     |
| `Alt + ,`                   | Открыть настройки                    |
| `Alt + R`                   | На главную                           |
| `Shift + Enter`             | Добавить задачу на следующий день    |
| `Shift + клик по +` (месяц) | Добавить цель в следующий день       |

### 🧩 Общие возможности

- Все данные в `localStorage`
- Защита от XSS (экранирование HTML)
- Всплывающие уведомления
- Адаптивный дизайн (мобилки, планшеты, ПК)
- Кастомный скроллбар
- Выделение текста запрещено везде, кроме полей ввода
